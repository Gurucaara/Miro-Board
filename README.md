## Getting Started

**Cloning the Repository:**

````bash
git clone [https://github.com/Gurucaara/Miro-Board.git]

### Install packages

```shell
npm i
````

### Setup .env file

```js
CONVEX_DEPLOYMENT=
NEXT_PUBLIC_CONVEX_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
LIVEBLOCKS_SECRET_KEY=
STRIPE_API_KEY=
NEXT_PUBLIC_APP_URL=
STRIPE_WEBHOOK_SECRET =
```

### Setup Convex

```shell
npx convex dev

```

### Start the app

```shell
npm run dev

```
